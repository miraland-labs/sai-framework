# SAI Framework Roadmap

This document outlines the planned development of the Sustainable AI Intelligence (SAI) Framework.

## Current Status (v0.1 - July 2026)

- Core 3D coordinate system defined
- Metrics and aggregation methods for X, Y, Z axes
- Sustainable Intelligence Index (SII) specification
- Basic evaluation workflow
- Initial Web3/tokenization guidance

**Files Published:**
- README.md
- SPEC.md
- CONTRIBUTING.md

## Near-Term Goals (v0.2 — Q3 2026)

- Reference Python library for SII calculation and visualization
- Example evaluation notebooks for major models
- Expanded benchmark mappings and category templates
- Tooling recommendations and integration guides
- Initial conformance testing suite

## Medium-Term Goals (v0.3 — v1.0 — Q4 2026)

- Formal governance model
- Web3 integration reference (on-chain score schema, oracle design)
- Sample NFT certificate implementation
- First reference implementations for popular inference frameworks
- Community-driven benchmark validation

## Long-Term Vision (2027+)

- v1.0 Stable specification
- Potential submission to standards bodies (e.g., Linux Foundation, IEEE, or ISO working group)
- Certification program for SAI-compliant evaluations
- Ecosystem of tools, dashboards, and leaderboards
- Widespread adoption in enterprise AI procurement and DeAI projects

## How to Influence the Roadmap

- Open an Issue with the label `roadmap`
- Participate in Discussions
- Submit Pull Requests for new features or improvements
- Join community calls (to be scheduled)

We welcome feedback on priorities — especially from practitioners in AI operations, sustainability, and Web3/DeAI.
