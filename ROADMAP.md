# SAI Framework Roadmap

Planned development of the Sustainable AI Intelligence (SAI) Framework specification.

## Current Status (v0.2.1 — July 2026)

**v0.2.1 errata / hardening:**

- Fixed partial-coverage Y formula; aligned worked example with reference calculator
- Clarified suite-level energy scope and probe-scaling rules
- Corrected decode-phase energy formula; added published rounding rules
- RFC-style MUST/SHOULD/MAY conventions; calculator `--self-test`

**v0.2 baseline:**

- Precise X, Y, Z definitions; three-tier energy; four workload categories
- Conformance levels; evaluation report schema; stdlib reference calculator

**Artifacts:** README, SPEC, BENCHMARK_GUIDE, MEASUREMENT_PROTOCOL, CONFORMANCE, schemas/, examples/

## Near-Term (v0.3 — Q4 2026)

- Web3 integration specification (on-chain schema, oracle design)
- Official GPT-3.5-turbo baseline dataset (replaces provisional Baseline_Y / token totals)
- Conformance checklist validation tooling (companion repository)
- Community benchmark validation working group

## Medium-Term (v1.0 — 2027)

- Stable specification
- Formal governance / neutral foundation transition
- SAI-Certified program with independent lab verification
- Carbon footprint certification pathway

## Long-Term (2027+)

- Standards-body submission (Linux Foundation, IEEE, ISO)
- Ecosystem of external conformant implementations and leaderboards
